How to install???
Now that's easy!
Use Windows 7 Start Button Changer to change the start orb. You can find this tool here:
http://www.door2windows.com/windows-7-start-button-changer-change-your-windows-7-start-button-with-one-click/
Launch Windows 7 Start Button Changer and choose the preferred BMP from this folder and you're done!
